# 팀 프로젝트명 carindrive		

- 소개 : 차량 렌트사이트 제작			  
- 기간 : 2023 06 20 ~ 2023 10 06
- 팀원 : 5명

- 개발환경과 사용한 기술
	- Windows
	- Eclipse 4.27 , STS3
	- Spring boot, MyBatis, JDK1.8
	- Oracle 18c, SQL Developer

 - 저의 역할
	차량예약, 포트원API활용 결제, 환불, 예약확인, DB설계, 관리자페이지 환불처리

● 경험한 것
- 사용자 입장에서 테스트해보고 기능을 설계 및 구현하였습니다.
- HTML, CSS ,Javascript를 이용하여 인터페이스를 구현하고 클라이언트의 동작에 따른 이벤트를 발생시켰습니다.
- 포트원API를 이용하여 결제 시스템을 구현했습니다. (실제로 돈이 빠져 나갔다가 자정 가까이 환불됩니다(TEST용))
- 오늘 날짜를 기준으로 환불 일자에 대한 환불금액 차등 지급을 구현했습니다.
- 사용자가 직접 환불하기를 눌러서 환불을 가능하게 구현했습니다.
- 관리자 페이지에서 소비자의 에러에 대한 요청이 있을시 
관리자가 직접 해당 고객의 결제를 환불 가능하게 만들었습니다.
  (예상한 구도는 1. 고객이 고객센터에 환불요청 작성 -> 2. 관리자가 확인 -> 3.환불처리)


● 고민한 것
- 대여날짜 선택시 현재시간보다 이전의 날짜를 선택할수 없게 자바스크립트로 구현했습니다.
- 예약내역이 많을때 최신순으로 정렬하기 위해 Collections을 통해 정렬후 반복문으로 꺼내어 출력했습니다.
- 포트원 API 를 사용하기 위해 주문번호와 token을 기준으로 결제, 환불이 원활하게 작동하게 만들었습니다.
- 사용자가 차량예약에 집중할 수 있게 인터페이스를 구현했습니다.
- 사용자가 환불할시에 보안요소로 비밀번호를 입력 후 환불이 가능하게 구현했습니다.
- 사용자가 차량을 예약중일때 해당차량을 동시에 예약할 수 없게 
해당 범위의 시간의 차량을 다른 사람은 5분동안 예약할 수 없게 구현했습니다.
  (DB쿼리문에서 EXTRACT 함수를 사용해서 렌트 가능상태 , 불가능상태 로구현했습니다.)
- 환불이 된것들은 환불이 안되도록 자바스크립트로 버튼을 안보이게 구현했습니다.


● 아쉬운 것
- 고객센터에 글이 작성될시에 관리자에게 문자나, 메일이 가게끔 확인을 유도하는 기능을 넣었다면 좋지 싶습니다.
- 차량을 예약할 때 날짜 범위를 선택하면 그에 맞는 가격이 비동기식으로 바로바로 나타나게 하는 방법이 더 좋았을 것 같습니다.
- 코드가 너무 길어서 코드를 조금 더 세세하게 나누었으면 유지보수에 용이했을것 같습니다.
